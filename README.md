Super-Mario

Jogo Super Mario desenvolvido no primeiro ano do curso Técnico em Informática, utilizando HTML, CSS e JavaScript.

Inspirado no jogo do dinossauro do Google, o jogador precisa pular obstáculos (canos) enquanto o cenário avança. O jogo também conta com um sistema de seleção de personagens, permitindo escolher entre o Mario e os 151 primeiros Pokémon.
